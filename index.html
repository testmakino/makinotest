Makino SAP Process Guide

Parts Return

Step 1: Go to VA01 (Create Sales Document)

Access the return order creation functionality through the standard sales document transaction.

Step 2: Select ZRT (Makino Returns)

Choose the return order type configured for the return process.

Step 3: Fill in Sales Org - 1010 (Makino Singapore)

Enter the correct organizational data for processing.

Step 4: Complete Return Details and Save

Include customer info, reference, reason codes, and material details.

Quotation

Step 1: Go to VA21 (Create Quotation)

Enter quotation type ZCS and sales organization data.

Step 2: Fill in Customer & Material Info

Include reference, model, materials, quantity, etc.

Step 3: Check Stock via MD04 / ME1M

Ensure lead time availability or update quotation accordingly.

Step 4: Save Quotation

Generate PDF and send it to the customer.
