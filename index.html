Quotation Process

  1. Go to VA21 (Create Quotation).
  2. Select Quotation Type: ZCS (Makino_Quot_Service).
  3. Enter Organizational Data: Sales Org 1010, Dist Channel 10, Division 99, etc.
  4. Enter Customer Reference and Date.
  5. Fill in Material Number, Quantity, Route.
  6. Select Incoterms, Payment Terms, and Usage.
  7. Go to MD04/ME1M to check stock.
  8. If needed, update lead time in VA22.
  9. Generate and output PDF.

Sales Order

  1. Go to VA01 (Create Sales Document).
  2. Enter Order Type: ZPO, Customer PO number, Requested Delivery Date.
  3. Enter Material Number, Quantity, Plant (1012), Route.
  4. Select Payment Terms, Incoterms.
  5. Verify Net Price, Save to generate SO Number.

Outbound Delivery

  1. Use VL01N to create outbound delivery with sales order ref.
  2. Enter Shipping Point: 1012.
  3. Update delivery date, print DO, and coordinate packing.
  4. Use VL02N for PGI and airway bill update.
